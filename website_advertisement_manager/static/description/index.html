Odoo Website Advertisement Manager

Odoo Website Advertisement Manager is a module in which several block positions are created on the website and the customers can purchase those blocks for the advertisement purpose and can add desired content. The block content will be published automatically according to the booking date of the block.

Basic Features

  • Advertisement can enhance the website by providing the block feature.
  • Admin provides a space for the customers wherein customers can add the request for the advertisement by selecting the required available option.
  • Admin has the right to set the price per day for every advertisement position.
  • The customer has the flexibility to purchase the advertisement display position according to the availability.
  • A customer can select the date range for which advertisement will be displayed and make payment accordingly.
  • Customers can add Banners on the advertisement position booked.
  • Admin can allow/disallow external link for banner images added by the customer for their advertisement.
  • Admin can configure the auto-approval settings for the block content.
  • Admin will have complete control of advertisements displayed on the website.
  • Admin can view the list of customers who have purchased the advertisement blocks.
  • Admin can approve/deny or delete the advertisement banners added by the customer.
  • A mail will be sent to the customer on the approval of block content.
